Instructions

To use the JSON Editor:

  1. Go to the JSON Editor page.
  2. Upload your JSON files using the file input.
  3. Specify the preset name and new chance value.
  4. Click "Update All Files" to apply changes.
  5. Download the updated files as a ZIP archive.

To use the Craft Generator:

  1. Go to the Craft Generator page.
  2. Enter the max tier (e.g., 3 for T3).
  3. Specify upgrade items and action names for each tier.
  4. Add classnames (manually or from a file).
  5. Click "Generate Files" to create recipe files.
  6. Download the generated files as a ZIP archive.
